# 官网组件清单（客户建议拆分）

> 来源：`2026海外官网项目材料梳理.xlsx` → sheet「官网组件清单」。
> 本文件是**客户方建议的组件拆分**，作为使用与配置手册的组织依据。
> 每个组件的截图已导出至 [`./assets/`](./assets/)，命名规则 `页面__组件__序号.png`（序号 2 一般为 M 端 / 变体样式）。
> 优先级：`P0` 最高（首批必做） → `P2` 最低。

## 说明

- **清单口径**：本表逐行还原 Excel 原始拆分，不做增删；组件名保留客户原文。
- **实现映射**：`对应 block` 列是客户组件 → 本仓库 `Universal-editor-demo/blocks/` 实际实现的对应关系；`—` 表示尚未确认或未实现。
- **手册进度**：详细「使用与配置手册」按页逐页补充，见本目录其他文档。当前已完成 **首页 5 个**（见 [`首页-使用配置手册.md`](./首页-使用配置手册.md)）。
